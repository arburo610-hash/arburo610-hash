## Как пользоваться
- Один пост — один блок.
- Для каждого указывать: площадку, номер/тему из плана, статус (готов / опубликован + дата), финальный текст.
- После добавления — закоммитить в GitHub. Тогда и Claude Code (я), и бот (после следующего перезапуска) видят актуальный статус и не предлагают уже сделанное.
